## openMenu

[openMenu](https://github.com/mrneo240/openmenu) is a game launcher and menu for the Sega Dreamcast, designed for use with [GDEMU](https://gdemu.wordpress.com/about/) and compatible optical drive emulators. Written in C for KallistiOS, it provides a full-featured UI for browsing and launching a disc image collection from an SD card.

**Features:**
- Catalogs 466+ known USA & PAL retail games with metadata and box art
- ~85 popular Japanese retail games with metadata and box art
- 27 ported Atomiswave games with metadata and box art
- Multiple UI modes: Grid, LineDesc, and GDMENU list
- Custom user themes with persistent settings
- VMU icon support via [Crayon VMU](https://github.com/protofall/Crayon-VMU)
- Higher compatibility game loading via GDMENU loader (GD-ROM + CDI)
- Manually correct bad serial info from Sega

A modern fork, [openMenu Virtual Folder Bundle](https://github.com/derekpascarella/openmenu-virtual-folder-bundle) by Derek Pascarella, builds on the original to add virtual folder organization, per-game VMU device support, cheat files, a multi-disc launcher, and an actively maintained cross-platform GD MENU Card Manager.
