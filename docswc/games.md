My current focus on game development, low-level graphics programming, and retro console homebrew.

## nuQuake

Quake 1 client and engine fully hardware-accelerated for the **Sega Dreamcast**. Focuses on squeezing every bit of performance out of the SH4 CPU and PVR GPU using custom vertex formats and GLdc fastpath/map_buffer rendering.

- [GitLab](https://gitlab.com/HaydenKow/nuquake)

## GLdc Fork

A bleeding-edge fork of Kazade's GLdc OpenGL implementation for Dreamcast, introducing **fastpath** vertex submission and **map_buffer**, techniques that allow vertex data to be written directly into hardware-ready formats, yielding significant FPS gains.

- [GitLab](https://gitlab.com/HaydenKow/GLdc)

## Integrity Engine

Open source game engine targeting Sega Dreamcast, Sony PSP, Windows, Linux, and Mac OS.
